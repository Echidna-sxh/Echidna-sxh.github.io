# 管道和重定向的区别

这两个命令都是对标准输出输入进行改变

  • 重定向是直接把输出转到文件去，重定向符号后面接文件
  • 管道后面应该接的是命令，不是文件。
  • 管道左侧命令的输出作为管道右侧命令的输入
